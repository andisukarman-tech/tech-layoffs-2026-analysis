# Tech Layoffs 2026 Analysis

# Overview
This project analyzes major tech layoffs in 2026, focusing on the impact of AI and stock market reactions.

# Objectives
- Identify trends in layoffs
- Analyze AI-driven job cuts
- Evaluate stock market response

# Tools
- Python
- Pandas
- Matplotlib
- Seaborn

# Key Insights
- 61% layoffs driven by AI
- Stock prices often increase after layoffs
- Shift toward AI-related roles
